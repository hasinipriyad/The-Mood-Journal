# Mood Journal

A minimal, dark-themed mood journaling app built with vanilla HTML, CSS, and JavaScript. Log how you're feeling each day, add a note, and revisit your entries anytime — no account required.

## Features

- **Mood selection** — choose from 5 emoji moods (great, happy, neutral, sad, exhausted)
- **Daily entries** — write a note alongside your mood and save it with a timestamp
- **Edit & delete** — update or remove individual entries, or clear everything at once
- **Persistent storage** — entries are saved to `localStorage` so they survive page refreshes
- **Responsive design** — works on mobile and desktop

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox)
- Vanilla JavaScript (DOM manipulation, localStorage)
- Font Awesome 7 (mood icons)

## Getting Started

No build step needed. Just open `index.html` in your browser.

```bash
git clone https://github.com/your-username/mood-journal.git
cd mood-journal
open index.html
```
